abarantia: a mini ~wanna-be~ blogging platform

## What is this?

This is a sketchbook where I can practice new concepts and new features from newer versions from the NextJS ecosystem.

## Tech Stack

- Next.js
- React
- Typescript
- Tailwind
- React Query
- SQLite3
